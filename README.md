# GhostMapX

基于 LSPosed 的系统级位置模拟模块，用于开发者调试位置信息程序的模拟工具。

A system-level location simulation module based on LSPosed, designed as a developer tool for debugging location-based applications.

Telegram: https://t.me/GhostMapLsp
